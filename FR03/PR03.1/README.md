# PR3.1

__Requirement:__
> The vision system shall detect colours in the hue ranges (red/yellow).

__Success Criteria:__
> Software corresponds to the hue ranges for red (355° - 10°) and yellow (51° - 60°) and the colours are detected.

__Method of Evaluation:__
> Place known objects in front of depth camera and run colour detection software.
